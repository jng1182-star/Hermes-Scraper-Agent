This is a self-contained web app. No installation required.

## Option A — Double-click (easiest)

1. Open **Finder** on your Mac
2. Navigate to this folder: `i_want_to_create_a_beautiful_social_20260523_041316_677/`
3. Double-click **`dashboard/index.html`**
4. The app opens in your browser automatically

That's it — no Terminal, no installation needed.

---

## Option B — From Terminal

```bash
open dashboard/index.html
```

## Troubleshooting

| Problem | Fix |
|---------|-----|
| Browser says "file not found" | Make sure you're opening `dashboard/index.html`, not a different file |
| Page is blank | Try a different browser (Chrome or Safari) |
| Graph doesn't render | Check browser console (F12) for errors |
